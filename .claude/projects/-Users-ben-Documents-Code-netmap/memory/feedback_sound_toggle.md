Sound/audio feedback in the TUI must have a toggle (on/off). User independently had the same idea for sound, confirming enthusiasm, but insists on the toggle.

**Why:** Sound in terminals can be disruptive in shared/office environments.
**How to apply:** Default sound to off, add a keybinding to toggle, and a --sound CLI flag.
